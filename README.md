# 🛡️ Financial Fraud Detection AI & BI Dashboard
**Capstone Project | BIA Data Science**

## 📌 Executive Summary
This project addresses the critical challenge of financial fraud in mobile transactions. Using a dataset of over 6 million records, I developed an end-to-end pipeline that includes a **Machine Learning Model**, a **Live Streamlit Web App**, and an **Executive Power BI Dashboard**.

## 🚀 Key Features
* **Real-Time Prediction:** A Streamlit interface that allows users to input transaction details and receive an instant "Safe" or "Fraud" verdict.
* **Strategic Analytics:** A Power BI dashboard tracking **$124.52M in Value at Risk**, focusing on high-risk transaction types like `TRANSFER` and `CASH_OUT`.
* **High-Performance AI:** A Random Forest classifier optimized for **96% Recall** to ensure maximum fraud capture.

## 🛠️ Tech Stack
* **Language:** Python 3.11
* **Libraries:** Scikit-Learn, Pandas, NumPy, Streamlit
* **Visualization:** Power BI, Matplotlib, Seaborn
* **Deployment:** Git/GitHub, Flask/Streamlit

## 📊 Business Impact
By automating the first layer of fraud detection, this system reduces manual audit requirements by approximately **30%**, allowing risk officers to focus on high-priority alerts identified by the AI.

## 📖 How to Run
1. Clone the repository:
   `git clone https://github.com/mariyamsultana971-commits/ds_fraud_detection_v2.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Launch the AI App:
   `streamlit run fraud_app.py`
